# Expense Logger Android App

A simple Android app to log daily expenses with:
- Amount
- Category
- Date
- Payment method

It stores entries in a local Room database and shows saved records on the same screen for quick recall and analysis.

## Build APK

```bash
./gradlew assembleDebug
```

Debug APK output path:

`app/build/outputs/apk/debug/app-debug.apk`
